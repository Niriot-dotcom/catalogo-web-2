Tamaños de referencia

El tamaño de la fuente se determina de manera dinámica a partir de las unidades REM, estas representan el tamaño de fuente del elemento raíz.

Esto significa que 1rem es igual al tamaño de fuente del elemento raíz, que para la mayoría de los navegadores tiene un valor predeterminado de 16px.
